Welcome

Welcome to MyChampionList. Inspired by MyAnimeList, a WebApp which allows users to search up any anime/manga and grade it. It can be accessed at the following link: MyAnimeList

MyChampionList is a Web Application which allows users (suposedly League of legends players) to have a place to make a list of their favorite champions.

A bit of reference was also taken from the official Riot Games League of Legends Champions WebSite: Riot Games - League of Legends - Champions

MyChampionList is a project that I made for the Web Programing course at the University of Sibiu. The project is made using Angular, the Riot Games League of Legends API and Firestore Database. The project is hosted on GitHub and can be accessed at the following link: MyChampionList
